СДЕЛАНО:

Сверстан адаптивный лендинг, используя HTML5, CSS3, jQuery. Создано адаптивное меню на чистом CSS3. В построении секций использовался flex-box и grid. Использовался jquery для написания секции с портфолии и слайдера в подвале. Сайт адаптирован для мобильных устройств под разрешение от 420 px. Плагин для слайдера брендов http://owlcarousel2.github.io/OwlCarousel2/ Использовался java script для выстраивания картинок по категориям.
